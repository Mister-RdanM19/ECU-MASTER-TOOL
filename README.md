## 📖 Deskripsi Proyek
**Mr.Rm19 ECU Engine Master** adalah perangkat lunak profesional yang dirancang khusus untuk manajemen data biner ECU (Engine Control Unit). Dengan antarmuka **High-End Industrial Design**, alat ini memungkinkan teknisi untuk melakukan diagnosa, optimasi performa, dan pengamanan file biner secara presisi.

---

##  Daftar Fitur Lengkap

Aplikasi ini mencakup seluruh modul kontrol mesin berikut:

### 1. 🛡️ Security & Identity (Keamanan & Identitas)
* **Auto Signature Injection:** Menanamkan tanda tangan digital "Mr.Rm19" ke dalam file biner untuk klaim hak cipta dan identitas tuning.
* **ECU Lock System:** Protokol penguncian keamanan file untuk mencegah modifikasi tidak sah, mengubah status menjadi *LOCKED* atau *SECURE*.

### 2. 🏁 Performance Tools (Optimasi Performa)
* **Dyno Test Simulator:** Melakukan simulasi tenaga untuk mendeteksi Horsepower (HP) dan Torsi berdasarkan analisis biner.
* **Warm-up Mode:** Mengaktifkan mode pemanasan mesin otomatis dengan target Idle tertentu (misal: 2.200 RPM) untuk suhu kerja oli yang optimal.
* **Fuel Consumption Analysis:** Algoritma cerdas untuk menghitung estimasi efisiensi konsumsi bahan bakar (km/L) pasca-tuning.

### 3. 🔍 System Analysis (Analisis Sistem)
* **Anomaly Detection:** Pemindaian mendalam pada struktur biner untuk mendeteksi sektor yang korup atau tidak sinkron (Integrity Check).
* **Battery Health Monitor:** Pemantauan tegangan aki (Voltage) secara real-time untuk memastikan stabilitas selama proses transfer data.
* **Project Manager:** Database internal untuk mengelola daftar proyek tuning (seperti CBR250RR Racing, Daily Eco, dll.).

### 4. 🖥️ Workspace & Visualizer
* **Hex Visualizer Stream:** Menampilkan data mentah biner dalam format Hexadecimal (1024 bytes pertama) secara real-time.
* **Live Console Log:** Terminal interaktif yang mencatat setiap aktivitas sistem dengan stempel waktu (Time-stamp) yang akurat.
* **Dashboard Overview:** Panel ringkasan status Signature, Security, Voltage, dan Efficiency dalam satu layar.

## Cara Penggunaan
`DOWNLOAD FILE EcuMaster.exe`
1.  **Persiapan:** Pastikan Anda memiliki file `.bin` yang akan dikelola.
2.  **Upload:** Klik tombol **+ UPLOAD BINARY** untuk memuat stream data ke dalam workspace.
3.  **Eksekusi:** Pilih fitur di Sidebar kiri (seperti *Dyno Test* atau *Auto Signature*) sesuai kebutuhan.
4.  **Monitoring:** Pantau status pada *Stat Cards* di bagian atas dan log pada *Console* di bagian bawah.
